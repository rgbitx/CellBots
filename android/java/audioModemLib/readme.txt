AudioModemLib for Android
tehaynes@google.com (Tom Haynes)

This is an audio modem library that supports plugable layer 1 and layer 2 modules.

Currently Supported Modules
Layer 1:
  1200 BUAD Audio Frequency Shift Keying using Bell202 (RX & TX)
  
Layer 2: 
  HDLC aka AX.25 (RX Only)
  
Layer 2 framing: (RX & TX)
  AX.25 framing, used in packet radio
